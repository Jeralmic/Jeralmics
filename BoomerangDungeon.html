Project Overview

Boomerang Dungeon is top down boomerang bullet hell

This project is a prototype, it features bugs and many unfinished elements

What I Worked On

  • Designed core gameplay mechanics, AI behavior and progression.
  • Created all art assets and UI elements.
  • Skinned and animated all characters.

Tech Used

Unreal engine 4, Mgear, ngskin tools & Maya

Challenges & Learnings

This project came with many challenges, the biggest being rigging. I taught myself how to rig and animate characters, learning the process from scratch to bring the game's movement and attacks to life.
