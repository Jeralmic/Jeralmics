Project Overview

CityFlow is a puzzle game where players build roads to guide traffic.

DISCLAIMER! I built this project for Hackharvard 2024- Ive updated the graphics of this project since then to be more presentable then what was actually submitted to judges.

What I Worked On

  • Designed gameplay loop with other designers.
  • Created a clean & readable aesthetic to ensure roads and city layouts were visually distinct.
  • Designed and developed all art and UI assets
  • Created road building mechanic.

Tech Used

Unreal engine 5, Maya, Figma & Blender

Challenges & Learnings

For this project my biggest challenge was quickly prototyping and refining a functional and intuitive road building mechanic.
