Project Overview

Infinite Roads is a puzzle game where players build roads to guide traffic efficiently through a city.

DISCLAIMER! I built this project for Hackharvard. I've updated the graphics since the initial submission to create a more polished, presentable version.

Key Accomplishments

  • Designed gameplay loop alongside other designers (Jacob Mandelbrot & Aakarsh Singh)
  • Developed a clean, readable aesthetic ensuring roads and city layouts were visually distinct
  • Designed and developed all art and UI assets
  • Created intuitive road-building mechanic
  • Rapidly prototyped and refined the core mechanic during the hackathon

Tech Used

Unreal Engine 5, Maya, Figma, Blender
