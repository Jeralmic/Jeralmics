Project Overview

TOTEM is a puzzle game where players control a rat totem, eating cheese scattered throughout mazes.

Key Accomplishments

  • Designed all levels
  • Worked alongside other designers to develop the core game mechanic
  • Created all art assets, including character models and environments

Tech Used

Unreal Engine 4, Maya & Blender

Reflections

For this project I wanted to challenge myself to create art that would use bold colors, simple shapes, and a clean minimalistic style that makes the game stand out visually.
